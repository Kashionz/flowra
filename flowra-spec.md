# 個人現金流模擬器 — 功能擴充規格

## 1. 視覺化升級

### 1.1 現況問題
- 手刻 SVG 圖表，沒有 hover、tooltip、無法 drill-down
- viewBox 寫死 760×260，手機需要橫向 scroll

### 1.2 技術選型
改用 **Recharts + shadcn/ui chart wrapper**
- 內建 ResponsiveContainer、Tooltip、Legend
- 走 CSS variables 主題化，跟 Tailwind 設計一致
- 體積比 Plotly / d3 小，符合日式極簡風格（可關掉預設 grid，只留淡虛線）
- Sankey 另用 `@nivo/sankey`，因為 Recharts 沒有

### 1.3 五個圖表

#### A. 月底現金趨勢（升級版）
- 互動：hover 顯示該月明細浮層（薪資、總支出、淨額、月底現金）
- 參考線：
  - `y = 0`：紅色虛線（破產線）
  - `y = 緊急預備金目標`：綠色虛線（例如 3 個月支出）
- 「今天」垂直虛線標記，自動依 `new Date()` 對到對應月份
- 點擊月份節點 → 下方明細表自動 scroll 並 highlight 該列

#### B. 收入 vs 支出（升級版）
- 沿用現有 grouped bar 設計
- hover tooltip 顯示具體數字
- 月淨額正/負小標籤畫在 bar 群組上方
- 點擊長條 → 切到該月「支出組成」鑽取畫面

#### C. 支出組成堆疊面積圖（新）
- 顏色帶從下到上：固定支出（房租、學貸）→ 浮動（生活、分期）→ 一次性
- 切換按鈕：「絕對金額 / 佔比百分比」
- 用途：一眼看出「支出 60% 是房租 + 生活，分期只佔 15%」這類洞察

#### D. Sankey 金流圖（新）
- 單張圖呈現整段試算期的金流走向
- 路徑：薪資 / 補貼 / 一次收入 → 總收入 → 房租 / 生活費 / 學貸 / 分期 / 一次支出 / 期末結餘
- 給整個期間用，不用每月一張

#### E. 月度熱力圖（次要）
- 12 個月 × N 年格子矩陣，顏色深淺對應月底現金
- 長期持有時才有意義，短期 7 個月不必做

### 1.4 共通互動規格
- Tooltip 樣式：白底、圓角 8px、陰影、字體跟頁面一致，不用 Recharts 預設
- 空狀態：試算月數 = 0 時顯示「請至少設定 1 個月」
- Loading：切換情境時 200ms fade 過場
- 無障礙：所有圖表 SVG 加 `<title>` 跟 `aria-label`

## 2. UX 與資料輸入

### 2.1 絕對日期取代 month index
- **現況問題**：`0=5月, 1=6月` 註解容易看錯，且無法跨年
- 所有 startMonth 改用 `YYYY-MM` 字串，UI 用年/月兩段式下拉
- 試算起始月由全域 `baseMonth` 控制，不再隱含「第 0 個月 = 5 月」
- 內部仍可換算成 index 餵給現有 `buildProjection`，計算邏輯不動

### 2.2 一次性項目分類 tag
- 分類選項：醫療 / 旅遊 / 禮金 / 稅務 / 3C / 社交 / 其他
- 每個分類一個顏色 + emoji（🏥 ✈️ 🎁 📋 💻 🍽️）
- 月度明細表的「一次支出」欄改成顯示前 2 個分類 chip
- 支出組成圖可切換到「按分類」視角

### 2.3 批次貼上解析
- 「📋 批次匯入分期」按鈕開 Modal
- 使用者貼純文字（一行一筆：名稱、本金、利率、期數、起始月）
- 用 regex 解析每行
- 解析失敗的行用紅底標出讓使用者修正
- 預覽通過後一次匯入

### 2.4 範本系統
預設範本下拉選單：
- 🌱 社會新鮮人（低薪、無貸、學貸）
- 🏠 有房貸租屋族（目前的設定）
- 👨‍👩‍👧 有小孩家庭（多生活費、補習費年支）
- 💼 自由工作者（不規則收入）

範本 = 完整 state，存在 `/lib/templates/` 下，一鍵帶入。

### 2.5 行動版面
- 現況 `mainGrid` 用 inline grid 寫死，手機會擠
- 遷到 Tailwind responsive class 統一管理
- 手機改單欄：設定區在上、圖表表格在下
- 圖表 ResponsiveContainer 高度自適應（桌機 320 / 手機 240）
- 月度明細表手機改 card list 而非 table

### 2.6 隱私模式
- 切換鍵把所有金額遮成 ★★★
- 快捷鍵 `M` 切換
- 給人看圖表時用

### 2.7 拖拉排序
- `@dnd-kit/sortable` 給 installments 跟 oneTimeItems 拉動排序
- 不影響計算結果（仍按 startMonth 排序），純粹整理感
- 列表很長時很有用

### 2.8 Inline 微調
- 數字欄位旁邊加 ↑↓ 按鈕做 ±1000 增減（手機特別有用）
- 月份輸入支援 relative 快捷：「下個月」「再下個月」
- 一次性項目可複製（買重複生日禮物的場景）

## 3. 儲存功能

### 3.1 資料模型原則
- **永遠帶 schemaVersion**，未來改 schema 才有 migration 路徑
- 結構分為四層：
  - `meta`（情境名稱、描述、建立/更新時間、baseMonth）
  - `basics`（薪資、房租、生活費等所有 scalar 欄位）
  - `oneTimeItems`
  - `installments`
- 之後若加新功能（如目標、投資），都當新 array 掛在 root，不動既有結構

### 3.2 Phase 1：localStorage（先做這個）
**自動存檔**
- `useEffect + debounce 1s` 監聽整個 state，自動寫進「編輯中」slot
- 不佔正式情境清單
- 下次開啟自動還原

**手動命名儲存**
- 使用者按「💾 儲存為情境」才寫入正式清單
- 上方 dropdown 切換情境
- 操作：list / save / load / rename / delete / duplicate

### 3.3 Phase 2：Supabase 多裝置同步
- RLS 鎖 `auth.uid() = user_id`，個人工具最簡權限模型
- 登入後跳出「上傳本地情境到雲端？」對話框
- **雲端 = source of truth，本地當 cache + offline buffer**
- 同步用 `updated_at` 做 last-write-wins（個人工具不需要 CRDT）
- 離線時繼續寫 localStorage，連線後 push

### 3.4 匯入 / 匯出 JSON
- 跨機器搬家最低成本方案，不用等 Supabase 上線就能做
- 匯出：下載 `cashflow-{name}-{date}.json`
- 匯入：上傳檔案 → 驗證 `schemaVersion` → 不符就跑 migration

## 4. 分享連結

### 4.1 兩種模式

| 模式 | 機制 | 適合場景 |
|---|---|---|
| URL 內嵌 | state → JSON → lz-string 壓縮 → base64url → query param | 快速分享、無後端依賴 |
| 短網址 | 存 Supabase 給 6 字元 slug | URL 太長、隱私性更好（state 不在歷史紀錄） |

### 4.2 URL 內嵌邏輯
- 即時生成，貼上後另一端解碼還原 state
- **長度檢查**：壓縮後超過 1800 字元就 fallback 短網址
- 實測目前 state（3 筆分期 + 2 一次性）壓縮後約 400–600 字元，遠在限制內

### 4.3 短網址邏輯
- 6 字元 nanoid slug
- 預設 30 天過期，過期後讀不到
- 公開讀取（非過期才讀得到），不需要登入就能看
- 計 `view_count` 做基礎 analytics

### 4.4 分享 UX
- 「🔗 分享」按鈕開 Popover，兩個 tab：
  - **複製連結**：URL 編碼版（即時，無需登入）
  - **短網址**：需登入，產生後顯示 + QR code
- 開關「分享為唯讀」：接收方不能編輯，但可以「複製成自己的版本」
- 警語：「⚠️ 連結內含你的金額資訊，僅分享給信任對象」

### 4.5 隱私模式分享
- 「★★★ 模式分享」：產生連結時把所有金額替換成 0，只保留結構
- 給人看你的試算邏輯而不是真實數字
- 適合炫技、發 blog、教學用

## 5. 圖表 / 報表下載

### 5.1 三種輸出對應三種場景

| 輸出 | 格式 | 用途 |
|---|---|---|
| 單張圖 | PNG | 貼 LINE / 簡報 |
| 完整報表 | PDF | 給會計、配偶看、自己存檔 |
| 明細資料 | XLSX | 自己再分析、跟其他工具串 |

### 5.2 PNG 匯出單張圖
- 每張圖右上角加 ⬇️ 按鈕
- 套件：`html-to-image`
- 解析度：`pixelRatio: 2`（retina 清晰度）
- **注意**：Recharts 用 SVG，渲染前必須 `await document.fonts.ready`，否則中文 webfont 會漏字

### 5.3 PDF 報表（一頁式）
內容由上而下：
- 標題 + 試算期間 + 產生時間
- 4 張摘要 stat 卡片（期末現金、期間最低、總收入、分期利息合計）
- 趨勢線圖 + 收支圖 + 支出組成圖
- 月度明細表
- 頁尾隱私聲明：「資料儲存於本機，未上傳」

實作流程：擷取整個 report 區塊為 PNG → 等比縮放塞進 A4。

### 5.4 XLSX 明細匯出
- 月度明細表逐欄輸出（月份、月初現金、薪資、補貼、各支出、淨額、月底現金）
- 套件：`xlsx` (SheetJS)
- 一個工作表先做就好，未來可加「分期清單」「一次性清單」工作表

### 5.5 列印樣式
- `@media print` CSS 隱藏所有按鈕、設定區
- 只留標題 + 圖表 + 表格
- 給不想下載 PDF 的人 `Cmd+P` 直接印

### 5.6 統一下載 UX
右上角放一個「⬇️ 匯出」dropdown：
- 📷 截圖（PNG，整頁）
- 📄 PDF 報表
- 📊 Excel 明細
- 🗂️ JSON 完整狀態（給匯入用）
- 🖨️ 列印
